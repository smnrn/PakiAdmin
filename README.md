# PakiADMIN (Copy)

This project now runs as a TypeScript React app on Next.js.

Original design source:
https://www.figma.com/design/DQDezmNT9l7QsZ5iUSCp6o/PakiADMIN--Copy-.

## Run locally

1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`
3. Open `http://localhost:3000`

## Production

- `npm run build`
- `npm run start`
